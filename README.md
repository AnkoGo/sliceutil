======

sliceutil 提供了针对数组和切片的功能

- Delete 删除符合条件的切片元素；
- QuickDelete 删除符合条件的切片元素，性能稍高于 Delete；
- Count 统计数组或切片中包含指定什的数量；
- Dup 查看数组或切片中是否包含重得的值；

安装
----

```shell
go get github.com/issue9/sliceutil
```
